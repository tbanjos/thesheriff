# RESTful API definition

## /v1/bandido

* Methods
  * GET
    * Content-Type: JSON
    * Parameters
      * id: integer, returns single element
      * none: returns all elements
    * Returns
      * Array Bandido(s) JSON
  * POST
    * Parameters
      * Bandido JSON
    * Returns
      * JSON success message
